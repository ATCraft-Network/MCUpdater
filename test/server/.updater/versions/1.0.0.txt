新特性：
- 全新冒险体验：加入了多个全新的冒险地图和任务链，探索未知的地下世界和远古遗迹。
- 魔法与科技的融合：新增了魔法与科技的跨界模组，玩家可以同时进行魔法研究和高科技设备的制造。
- 自定义物品：加入了自定义的装备、物品和食物，丰富了游戏的多样性和策略性。
- 多人联机支持：对多人联机进行了优化，减少了延迟和卡顿，玩家可以无缝地进行合作或对战。

修复与优化：
- 性能优化：通过调整资源加载机制和优化内存占用，显著提升了整体游戏性能，减少了大规模生物群落和复杂建筑的卡顿问题。
- 模组兼容性：修复了与一些流行模组（如 Biomes O' Plenty 和 Tinkers' Construct）之间的兼容性问题，确保它们可以正常共存。
- UI改进：对游戏内的菜单和界面进行了优化，提升了用户体验，尤其是在物品管理和工艺界面的响应速度。

新增模组：
- "MysticCraft"：一个全新的魔法模组，玩家可以通过灵气、符文和炼金术来提升自身能力，开启神秘的魔法之旅。
- "TechNova"：一款专注于高科技发展的模组，提供了新的机器、自动化系统和能源管理方案，帮助玩家打造超级先进的科技帝国。
- "ShadowRealms"：一款全新的维度模组，引入了神秘的暗影领域，玩家可以进入该领域进行挑战，寻找强力装备和神器。

平衡调整：
- 生物生成：减少了某些过于强力的敌对生物的生成频率，增加了更多类型的敌人，挑战性更高。
- 资源掉落：调整了部分资源的掉落机制，确保在冒险过程中更公平和合理。

已知问题：
- 部分 mod 的配置文件可能需要手动更新，以避免与新版本的整合包发生冲突。
- 在极少数情况下，首次生成的世界可能会出现临时的地形异常，建议重新生成世界。
